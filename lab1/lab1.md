# Reflection for Lab 1

The thing i like about python is that the syntax is very simple. I do not need to worry about semicolon, brackets and data types. It makes coding very easy to write and looks cleaner.

Something that behaved differently was how loops are ranges work in python. For example, the ending number in range() is not included. 

Python and C/C++ are similar because they both use functions, loops, and classes. The logic for the solution is mostly the same. However, Python is much more simple and shorter to write. In C/C++, I usually need to declare variable types and use braces and semicolons, while Python manages those automatically. Because of this, writing programs in Python feels faster without worrying about typical syntax errors.
